# Pé de Serra — Cafés Especiais

Site institucional da marca Pé de Serra, especializada em cafés especiais das regiões do Caparaó e Matas de Minas.

## Tecnologias

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Desenvolvimento local

Pré-requisito: Node.js & npm — [instalar com nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Clonar o repositório
git clone <URL_DO_REPOSITORIO>

# Entrar na pasta do projeto
cd serra-sweet-site

# Instalar dependências
npm i

# Iniciar servidor de desenvolvimento
npm run dev
```

## Build de produção

```sh
npm run build
npm run preview
```
